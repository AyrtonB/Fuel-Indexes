# Global Gas Prices

<br>
 
### Set-Up

Start by installing Anaconda, then open Anaconda Prompt and run the following commands
```bash
cd "path/to/Global-Gas-Prices"
conda env create -f environment.yml
ipython kernel install --user --name=Gas
```
